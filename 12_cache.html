У фреймворці передбачена система кешування. Кешувати можна любі дані. Результат обрахунків, хтмл, обєкти, результат вибірки з бази даних.

Перед роботою з кешем потрібно настроїти драйвер.

Існують 3 драйвера, які можна вибрати і настроїти через файл config/params/param.php

'cachedriver' => 'fileCache'

по замовчуванні кеш зберігається у файлові системі

тут потрібно настроїти параметр

'filecachePath' => 'cache'

це шлях до папки де будуть зберігатися кеші у вашому проекті

також можна вибрати драйвера або redisCache, або memCache

для

'cachedriver' => 'redisCache'

потрібно настроїти наступні параметри

'redisHost' => '127.0.0.1', 'redisPort' => '6379', 'redisPass' => 'foobared',

для

'cachedriver' => 'memCache'

потрібно настроїти наступні параметри

'memcacheHost' => '127.0.0.1', 'memcachePort' => '11211',

Для добавлення даних в кещ потрібно

$container->get('cache')->set('key', $data);

в кеш під ключем кеш добавляться дані $data

$data має бути масивом, це головне, а дані які будуть в масиві можуть бути які вам потрібно

$container->get('cache')->set('key', ['tpl' => '<p>tets</p>', 'dbres' => $products]);

для отримання даних з кешу потрібно

$data = $container->get('cache')->get('key');

в даному випадку

$data = ['tpl' => '<p>tets</p>', 'dbres' => $products];
