Для інсталяції потрібно виконати з консолі

composer create-project leshko/la-skeleton:dev-master ваша_папка

Після цього зайти в папку проекту і виконати

composer update

Дальше зкопіювати файл config/params/param_dist.php у config/params/param.php

Дальше зайдіть у папку template і виконайте

bower install bootstrap
